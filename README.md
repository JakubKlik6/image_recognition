# Django AI  
Requirements:
- Django 4.0
- Pillow
- numpy
- pandas
- tensorflow 2.8.0
- tensorflow-hub
- protobuf 3.20

## Install
```sh
pip install -r requirements.txt
```

## Lunch
```sh
python manage.py runserver
```

# Description
This is a simple Django application in which after uploading any photo, using gpt chat, it creates a description and title for the image.
